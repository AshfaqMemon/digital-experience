# Modify media assets 

Learn how to modify uploaded/stored media assets using HCL Digital Asset Management.

-   **[Edit image ](../digital_asset_mgmt/dam_edit_image.md)**  
 HCL Digital Experience comes with functionalities to edit media assets.
-   **[Edit the metadata of a media asset ](../digital_asset_mgmt/dam_metadata_properties_media_asset.md)**  
 This section shows how users to edit the embedded metadata of a media asset in HCL Digital Asset Management.
-   **[Generate renditions and versioning ](../digital_asset_mgmt/dam_generate_renditions_and_versions.md)**  
 Learn more how to view, edit, and manage media asset renditions and versions with HCL Digital Asset Management.
-   **[Set unique custom URL for a media asset ](../digital_asset_mgmt/dam_set_unique_custom_URL_media_asset.md)**  
 The media assets in the HCL Digital Asset Management can be accessed through a URL. However, this URL is long and complex since it contains unique identifiers to locate the resource. Digital Asset Management allows users to customize complex URLs and change them to human-readable URLs, which are shorter, readable, and easier to use.
