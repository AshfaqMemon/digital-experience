# Combined Cumulative Fix Strategy

The Combined Cumulative Fix is a package of HCL Digital Experience fixes. Cumulative fixes can only be applied to an existing functional version of HCL Portal Version or an HCL Portal Version with a previous cumulative fix applied.

Refer to the [Combined Cumulative Fix strategy](https://help.hcltechsw.com/digital-experience/8.5/overview/ccf_strategy85.html) instructions to learn how to apply or roll back a combined cumulative fix for HCL Digital Experience 9.5 and later releases.
